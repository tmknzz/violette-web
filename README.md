# Violette website

Marketing site for **Violette — communication helper for macOS**.

- Live site: https://tmknzz.github.io/violette-web/
- Product: https://tmknzz.lemonsqueezy.com/checkout/buy/c8221ccf-1de8-416c-818a-a99c8e4ae105
- App source (private): separate repo

Static HTML, no build step. Publishes automatically via GitHub Pages from `main`.

## Contents

- `index.html` — landing page
- `terms.html` / `privacy.html` / `refund.html` — legal pages
- `style.css` — shared styles
- `violette-icon.png` / `violette-full.png` — app icon assets

## DMG distribution

Notarized DMG builds are uploaded as GitHub **Releases** on this repo. The site's Download button links to `/releases/latest`.
